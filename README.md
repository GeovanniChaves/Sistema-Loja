# 🛒 Sistema de Loja em Java

## 📌 Sobre o projeto

Este projeto consiste em um sistema simples de gerenciamento de produtos desenvolvido em **Java**, com execução via console.

O objetivo foi praticar os fundamentos da linguagem, trabalhando com estruturas de controle, entrada de dados e manipulação de arrays.

O sistema permite realizar operações básicas de gerenciamento de produtos, simulando um pequeno CRUD (Create, Read, Update e Delete).

---

## 🚀 Funcionalidades

* ✅ Cadastro de produtos
* ✅ Listagem de produtos cadastrados
* ✅ Busca de produtos pelo nome
* ✅ Remoção de produtos
* ✅ Controle de quantidade de produtos cadastrados
* ✅ Menu interativo no console

---

## 🛠️ Tecnologias utilizadas

* Java

* Estruturas condicionais (`if/else`)
* Estruturas de repetição (`for` e `while`)

## 💻 Como executar o projeto

### Pré-requisitos

Ter instalado:

* Java JDK 17 ou superior

### Executando

1. Clone o repositório:

```bash
git clone URL_DO_REPOSITORIO
```

2. Abra o projeto em uma IDE Java, como:

* Eclipse
* IntelliJ IDEA
* VS Code

3. Execute a classe:

```text
Loja.java
```
